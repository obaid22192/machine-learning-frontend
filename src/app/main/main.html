Mirabeau Machine Learning

The application to easily apply machine learning.
Designed for Mirabeau.

storage

Use your own datasets to experiment! You can upload and apply machine learning on the datasets you choose. The application will then run machine learning for you.

tune

The application provides you with several algorithms. You can choose which algorithms you want to use! Can't decide? Choose multiple and compare the results!

history

Want to compare old experiments? You can! The history of experiments will be saved, so you can run them again and compare the results.
